世界崩壊少女『初希アイ』

プロジェクトサイトへ
プロジェクト概要

人類が滅亡した世界で唯一生き残った少女「初希アイ」の世界を、 OpenAIとXの力で実現する創作プロジェクトです。

技術スタック
  • HTML / CSS / JavaScript
  • Flask / Python
  • SQLite
  • OpenAI GPT（文章） / DALL·E 2（画像）
  • Google Vision API（画像検査）
  • Lolipop（レンタルサーバー）
ユーザー体験の流れ
  1. WEBサイトにアクセス・投稿
  2. 投稿内容がDB保存される
  3. Cron + PythonでAIがポスト内容を生成
  4. Xに自動投稿
  5. ユーザーが結果を楽しむ
特徴・こだわり

初希アイのXアカウントを見たとき、まるでその世界に生きる少女本人が投稿しているかのように見えることを重視しました。そのため、AI出力によるブレが表に出ないよう、生成結果に対して評価・改稿・添削の処理を徹底し、世界観との一貫性を保っています。

課題・工夫

ユーザーの投稿内容を尊重しつつ、世界観やキャラクター設定に沿った出力を実現する評価ロジックの構築に苦戦しました。プロンプト内の指示だけでは生成精度の調整が難しく、OpenAI APIの応答に対して独自の採点方式を実装することで、より安定した出力を可能にしました。現在も品質向上のためのアップデートを続けています。
また、GoogleVision APIによる画像解析処理では、DALL·E2が生成したPNG画像をそのまま処理していたため、メモリ使用量がサーバーの上限を超えるエラーが発生しました。エラー内容が一貫せず、Tracebackの出力も毎回異なっていたため、当初は原因の特定に時間を要しました。最終的には、画像形式の変換によりメモリ負荷を抑えることで、安定した処理を実現しました。

制作目的

AIにも人間に近い創作ができることを実証したいという好奇心から、実験的な取り組みとして制作をスタートしました。AIはあくまで与えられた指示の範囲内でしか生成できませんが、そこにランダムなプロンプトを加えることで予測不可能な要素が生まれ、観察する人間側には、あたかも意思や感情を持った存在が創作しているかのように感じられます。

X（旧Twitter）のアカウントを舞台にするアイデアは、約20年前にネット掲示板で見かけたある書き込みにヒントを得ました。そこには「オンラインゲームのフレンドに会いに行ったら相手がブリキのおもちゃだった」という夢の話が綴られていました。実際、SNSのタイムラインに流れる投稿をしたユーザーが、同じ三次元空間に生きる人間だとは誰にも断言できません。この曖昧さを活かし、人間の想像力を利用することで、別世界からの投稿として楽しむコンテンツを目指しました。
