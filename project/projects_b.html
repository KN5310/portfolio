きのこたけのこSNS

プロジェクトサイトへ
プロジェクト概要

SNSにおける“対立感情”や“承認欲求”を、ゲーム要素の原動力として活用した国家対立型の独自SNS体験を提供する創作プロジェクトです。

技術スタック
  • HTML / CSS / JavaScript
  • Flask / Python / Jinja2
  • MySQL
ユーザー体験の流れ
  1. 所属国のユーザー同士でタイムラインを通じて交流
  2. リアクションを届けるミニゲームで通貨を獲得
  3. ショップで便利な機能を解放
  4. 獲得した通貨で「文明」に投資
  5. 自国の魅力を強化し、他国と差をつける
特徴・こだわり

ユーザー登録時に選んだ“所属国”に応じてタイムラインを分離し、国家間の競争を促す設計が特徴です。アプリ内通貨や文明投資など、ゲーム的要素をSNSのコミュニケーションに融合させ、参加意欲や継続性を高める体験設計を重視しました。

課題・工夫

対立や競争を促しながらも攻撃的・排他的にならない、ユーモラスなデザイン設計で統一することに苦心しました。ポストに使用できる文字はひらカナ英数字に厳選し、絶えずシュールな雰囲気を忘れないようにしました。世界観やアニメ風にディフォルメし、ユーザーの分身となるキャラクターはちびキャラにしました。

制作目的

「自分好みのSNSを作ってみたい」という構想を、プログラミング学習の一環として形にしました。既存SNSの単なる模倣ではなく、“ゲーム性”や“社会性”といった要素を取り入れ、独自の設計を意識しています。

元々は、他のユーザーにゲーム的な攻撃ができるSNSとして赤組白組に分かれて楽しむ運動会を舞台にしたSNSを構想していましたが、アイコンの仮画像として使っていたきのこのイラストがきっかけで、「きのこたけのこ戦争」をテーマにした国家対立型SNSへと路線変更しました。
