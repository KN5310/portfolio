プロジェクト概要

妻のイラストを紹介するポートフォリオ兼オープニングロールページです。

技術スタック
  • HTML / CSS / JavaScript
  • Flask / Python
ユーザー体験の流れ
  1. ブログへのアクセス前にまずイラストを見る
  2. オートスクロールで深海に潜航する気分に浸りながらイラストを楽しめる
  3. 画面に波を起こして海の中にいる気分を高められる
  4. BGMを流すことができる
特徴・こだわり

妻の世界観を深海への潜航というコンセプトで表現しました。イラストをただ並べるのではなく、オートスクロールや波のエフェクト、BGMなど、没入感を高める要素にこだわりました。スクロールバーにも妻の大好きなめんだこアイコンをあしらい、細部にまで愛情を込めています。

課題・工夫

滑らかなオートスクロールの実現に手こずりました。波エフェクトは端末の処理力をみて解像度を調整する処理を仕込んであります。

制作目的

妻のイラストを多くの人に見てもらいたいという思いから制作しました。当初はイラストをただ並べるだけのシンプルなアトリエページを考えていましたが、妻の世界観をより深く伝えるために、深海への潜航というコンセプトを取り入れた没入感のあるデザインにしました。
