プロジェクト概要

令和7年全国アビリンピック、ホームページ部門の事前課題作品です。宮城県にある架空の街「千羽市」を舞台に、防災情報の啓発を目的としたWEBサイトを制作しました。公的な一次情報を正確に紹介し、ユーザーが必要な防災情報にアクセスしやすいデザインを心がけました。

技術スタック
  • HTML / CSS / JavaScript
ユーザー体験の流れ
  1. 数ある防災情報の中から推奨される防災情報を知ることができる。
  2. 防災リーダーやBPOなど、防災意識の中でも特に重要なことを知ることができる。
  3. 千羽市の被害状況や団体発足の経緯を知ることができる。
特徴・こだわり

防災情報の啓発という依頼の趣旨に添い、公的な一次情報を正確に紹介することを重視しました。架空の団体と地域を想定し、実際にありそうなリアリティを追求しました。

課題・工夫

団体の経験則に基づく言葉と、公的な一次情報のバランスが課題でした。アクセシビリティは基本を徹底しました。日々の更新作業は団体のメンバー（非エンジニア）が継続するものと想定し、ページの追加を単純化しました。

制作目的

令和7年全国アビリンピック、ホームページ部門の事前課題作品として制作しました。防災情報の啓発を目的とした団体のWEBサイトを想定し、正確な情報提供と分かりやすいデザインを心がけました。
